Tabli 4.0.1
Oct. 12, 2024
What's New (4.0.1):

The Tabli popup window is back!
Last month I was forced to ship an update with an unfortunate side effect: it removed the in-browser popup window.
Feedback from the community was swift and clear: the popup window is a critical feature for many users.
I'm pleased to report that, after a lot of painstaking work, the popup window is back!

Other Changes:
  • Tab De-Duplication now works for Google Docs URLs (Contributed by community member Killian Murphy)
Reminder: If you like Tabli and want to support its development, please take a moment to Review it on the Chrome Web Store. Thank You!
