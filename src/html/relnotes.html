Tabli 4.0.3
Oct. 25, 2024
What's New (4.0.3):
  • Quick fix to drag-and-drop tab re-ordering
What's New (4.0.1):
The Tabli popup window is back!
Last month I shipped an update to confom to new Chrome Extension requirements. An unfortunate part of that update was removal of the in-browser popup window.
Feedback from the community was swift and clear: the popup window is a critical feature for many users.
With today's update I'm pleased to report that, after a lot of painstaking work, the popup window is back!
Other Changes:
  • Tab De-Duplication now works for Google Docs URLs (Contributed by community member Killian Murphy)
Reminder: If you like Tabli and want to support its development, please take a moment to Review it on the Chrome Web Store. Thank You!
