Tabli 3.0.1
Sep. 14, 2024
What's New (3.0.1):
This release updates Tabli to the latest extension manifest format (V3) and a service worker, a mandatory requirement for all Chrome extensions.

As a consequence of this update, Tabli now only supports a free-standing popout window. While this has been the recommended way to use Tabli for many years, until now Tabli also continued to support an in-browser popup window.

Unfortunately, continuing to support the in-browser popup window would have required a complete rewrite of Tabli's internals, which was not feasible.

I did not make this change lightly, and explored all reasonable options for avoiding this change. I apologize for any inconvenience this may cause.
Other Changes:

  • Added a confirmation modal when un-managing a Saved Window
  • Added a link to the Tutorial Medium Article to the main menu
Reminder: If you like Tabli and want to support its development, please take a moment to Review it on the Chrome Web Store. Thank You!
